## **About me**

I am a Software Developer focused on backend development, system integrations, fiscal technology, APIs, cloud environments and observability.

I work with the development, maintenance and evolution of corporate systems, mainly involving fiscal document processing, REST and SOAP integrations, XML and JSON payloads, microservices, containers, monitoring and production support.

My experience includes projects related to **NFS-e, NF-e, CT-e, MDF-e and DC-e**, working with document emission, consultation, cancellation, fiscal events, payload validation, external APIs, web services and troubleshooting.
